macOS 首次打开

免费构建没有 Apple Developer ID 公证，macOS 可能提示“已损坏，无法打开”。安装到 /Applications 后运行：

sudo xattr -dr com.apple.quarantine "/Applications/Codex Desktop 管理器.app" && open "/Applications/Codex Desktop 管理器.app"
